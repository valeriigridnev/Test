Реализовать хэш-таблицу с функцией хэширования X % N, где X — целое число, помещаемое в хэш-таблицу, 
N — целое число, от деления на которое берётся остаток.
Коллизии разрешаются с помощью односвязного списка.
Реализованная программа должна выполнять добавление в хэш-таблицу элементов, заданных во входной последовательности.
После ввода всех элементов программа выводит на экран содержимое хэш-таблицы.
Использование реализаций списка из стандартной библиотеки ( list, ArrayList, LinkedList, Hashtable и т. п.) не допускается. 
В ходе решения должна быть получена своя реализация хэш-таблицы и односвязного списка с указанными интерфейсами.
